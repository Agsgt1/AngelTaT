# Habitat
Habitat is a collection of JavaScript helpers that I use in my hobby projects.<br>
You can embed it, like this:
```html
<script src="habitat-embed.js"></script>
<script>
  const { print } = Habitat
  print("Hello world!")
</script>
```
Or import it like this:
```javascript
import { print } from "./habitat-import.js"
print("Hello world!")
```
